# 迷你开发者工具箱 - 音乐脚本编辑器

> 为迷你世界开发者而生的工具类APP

## 概述

**迷你开发者工具箱 - 音乐脚本编辑器**可以帮你简化游戏开发过程，提供了可视化音乐编辑器，简化你的音乐创作过程。

源码已上传至[GitHub](https://github.com/tatsukimengchen/MusicScript/)

## 特性

* 可视化音乐编辑器
* 一键生成音乐脚本
* 多平台通用
* 小体积
* 简洁
